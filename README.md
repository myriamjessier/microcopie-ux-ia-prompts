# Bibliothèque de prompts — Microcopie UX × IA

Ressource du cours de microcopie UX de **Myriam Jessier**.  
36 prompts prêts à copier-coller pour travailler la microcopie d'interface avec Claude ou ChatGPT.

---

## C'est quoi ?

Une page web avec tous les prompts vus en cours, organisés par type de tâche.  
Pas besoin de chercher dans tes notes — tout est là, avec un bouton "Copier" sur chaque prompt.

## Comment l'utiliser

1. Ouvre la page → [myriamjessier.github.io/microcopie-ux-ia-prompts](https://myriamjessier.github.io/microcopie-ux-ia-prompts)
2. Navigue par catégorie dans le menu à gauche
3. Clique **Copier** sur le prompt qui t'intéresse
4. Colle dans Claude ou ChatGPT
5. Remplace les parties entre `[CROCHETS]` par ton contexte

## Ce que tu trouveras

| Catégorie | Ce que ça couvre |
|---|---|
| **Fondation** | Charte de voix, calibration depuis l'existant, scaffold XML |
| **Par type** | Erreurs, CTAs, états vides, onboarding, formulaires, confirmations, infobulles, 404 |

| **Bonus** | Prompts partagés par la communauté UX writing |

## Conseil pour bien démarrer

**Commence toujours par le prompt "Charte de voix"** (section Fondation).  
Tu le construis une fois pour ton projet, tu le colles en début de chaque session — c'est ce qui fait la différence entre un résultat générique et un résultat publiable.

## Conventions françaises intégrées

Tous les prompts respectent par défaut :
- Majuscule sur le premier mot uniquement (titres et boutons)
- Verbe à l'infinitif sur les boutons
- Vouvoiement ou tutoiement à préciser une fois, cohérent partout
- Typographie française (espaces insécables, guillemets «  »)
